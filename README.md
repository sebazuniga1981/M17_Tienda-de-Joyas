Desafío Tienda de Joyas – API REST
Descripción

Este proyecto corresponde al desafío Tienda de Joyas de Desafío Latam.
El objetivo es crear una API REST usando Node.js, Express y PostgreSQL para consultar joyas desde una base de datos, aplicando filtros, paginación, ordenamiento y estructura HATEOAS.
